# hello-vue
## 1. 프로젝트 새로 생성하는 경우
1-1. 프로젝트 생성
```bash
npm create vuetify@latest
```

1-2. 패키지 Install
```bash
npm install axios --save
npm install vue-router
```


1-3. 개발 모드로 애플리케이션 실행
```bash
npm run dev
```

## 2. 프로젝트 Clone 하는 경우

2-1. Node.js 패키지 설치
``` bash
npm install
```
2-2. 개발 모드로 애플리케이션 실행
```bash
npm run dev
```
